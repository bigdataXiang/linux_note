# linux下mongodb的安装
+ 在终端输入"mongo"
  如果系统没有安装mongodb，会提示应用
  "sudo apt install mongodb-clients"命令安装
+ 在终端输入"mongod"
  会提示应用命令"sudo apt install mongodb-server"
  安装
+ 完成上述的安装命令之后，执行"mongod"命令，打开mongodb服务
+ 执行"mongo"命令，查看数据库中的数据
